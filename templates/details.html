Voice Biometric Attendance System

                     How to Use System
    To use Registration with Cadence:
    1). On the Login page, click the “ I’m new here” option at the left bottom of the page. And you will be directed to the ‘Registration’ page.
    2). Once you are on the ‘Registration’ page enter the information carefully, which includes:
      • First Name , Last name , Phone number , Email , Post, Date of birth
      Click next and enter the information:
      • Address, Landmark, City, State, Latitude, Longitude, Pin code
      Click next :
      • Select a profile picture
    3). Click on the button ‘click here to record first voice sample’ in the same way , record the three samples.
    Note: You will be directed to the login page after submitting samples.

    To use Login with Cadence:
    1). Select your method of attendance.If you select geolocation the system will atomatically fetch users location.
    2).Click 'next' and speak the number clearly provided by the system.
    Note: Once the system recognises user, you will be automatically directed to the dashboard.

                       About System
    Cadence is a software specially developed for taking attendance through voice recognition. This web based software will work with GPS system/Geo-location system for its authenticity . It will work for all the officers working in Nehru Yuva Krida Sangathan at different districts and states. Candence provide user friendly and easy to understand platform to the user which will make the daily attendence system simple but efficient.
