Voice Biometric Attendance System

                     How to Use System
    Registration Process:
    1). Once signing in into the system, admin has option to Add User.
    2). Admin has to fill the registration form which includes folowing information about user:
      • First Name , Last name , Phone number , Email , Password, Date of birth
      • Address, City, State, Latitude, Longitude, Pin code
    3). After clicking on submit , user is successfully added.
    Login Process:
    1). If you are here for the first time, click on "Register voice sample" , else click on "Geo Location".
    2).Now you will be redirected to a page where you have to submit 3 voice samples .
    3). When you click on Geo Location option , browser will automatically detect the latitude and longitude .
    4).Click 'next' and speak the number clearly provided by the system.
    Note: Once the system recognizes user, you will be automatically directed to the dashboard.

                       About System
    Cadence is a software specially developed for taking attendance through voice recognition. This web based software will work with GPS system/Geo-location system for its authenticity . It will work for all the officers working in Nehru Yuva Krida Sangathan at different districts and states. Candence provide user friendly and easy to understand platform to the user which will make the daily attendence system simple but efficient.
